# E-commerce Analytics AI (Germany)

## Project Overview
This project analyzes e-commerce transaction data and builds an AI-powered assistant that generates SQL queries and business insights.

## Business Goal
Provide insights into sales performance, customer behavior, and product trends in an e-commerce environment.

## Dataset
Online Retail dataset (EU-based transactions)

## Key Metrics
- Revenue
- Average Order Value (AOV)
- Customer retention
- Top products and countries

## Tech Stack
- SQL
- Python (pandas)
- Data visualization
- LLM (AI assistant for query generation)

## Project Structure
data/        -> raw dataset
notebooks/   -> analysis in Python
sql/         -> SQL queries
app/         -> AI assistant (LLM integration)
## Status
In progress
